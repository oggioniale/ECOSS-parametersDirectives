CNR-IREA

June 8, 2021

The tools allow to:

  1. Evaluate the eLTER and Natura 2000 (N2K) sites’ contribution to the Marine Strategy Framework Directive (MSFD) monitoring activity (Directive contribution)

  2. Evaluate the N2K sites’ contribution to the conservation of target species and habitats included in the site (Conservation strategy)

1st use case

The first use case is designed for policy maker or reaserchers that want to assess the contribution of a monitoring site (e.g. Long Term Ecological Research (eLTER)) to the Marine Strategy Framework Directive (MSFD).

Directive contribution tool

Click on the first arrow to access the first tool about the contribution to the Marine Strategy Framework Directive (MSFD) monitoring.

Directive contribution tool

The graph show the site contribute to the MSFD. The parameters listed with the green checked circle represent the parameters monitored in the site that match with those deemed necessary to contribute to the MSFD criteria. The dark blue are all the MSFD criteria to which the parameters can contribute. The lines are meant to represent the connection between the parameters and the criteria.

2nd use case

The second use case is designed for those managers of Natura 2000 network site that want to assess the contribution of the site, to the species- or habitat-conservation strategy in order to implement and/or build a Management Plan.

Conservation strategy tool

Click on the second arrow to access the second tool about the evaluation of the conservation strategy.

Conservation strategy tool

This graph shows the site contribution to the conservarion strategy. Checked circles represent the recommended variables monitored in the site, while the light-blue symbol represents a variable that is not measured. The target species protected in the site are represented by the orange icon. Finally, also the protected habitats is/are shown.

Target species and habitat list are retrieved in real time from the European Environmental Agency (EEA) Linked Data repository.
